# Code of Conduct

## Standard

This project is intended to be useful, practical, and respectful for people who
want to learn or build AI-assisted branding websites.

Contributors are expected to:

- Be clear and constructive in issues, discussions, and pull requests.
- Critique code, product decisions, and documentation without personal attacks.
- Avoid spam, harassment, discrimination, or misleading claims.
- Respect user privacy and do not submit secrets, private customer data, or
  copyrighted assets without permission.

## Enforcement

Maintainers may edit, hide, or remove contributions that violate this standard.
Repeated or severe violations may result in being blocked from participating.
