# Marketing Experiment Backlog

기준일: 2026-05-27

## 사용법

이 문서는 수익이 날 때까지 반복할 마케팅 실험판이다. 실험은 한 번에 많이 하지 않는다. 매주 1~2개만 실행하고, 답장률과 상담 전환을 기준으로 유지/수정/중단을 결정한다.

| 3        | 스마트플레이스 + 홈페이지 연결 제안 | 학원/클래스 | 플레이스는 있지만 홈페이지가 없는 곳은 체험수업 CTA 개선에 관심이 있다 | 상담 1건        |

## 아웃리치 DM 템플릿

### 1인 전문가

```text
안녕하세요. 프로필과 콘텐츠를 보고 연락드렸습니다.

현재 소개, 후기, 상담 링크는 잘 쌓여 있는데 처음 보는 사람이 한 번에 이해할 수 있는 공식 페이지가 있으면 문의 전환이 더 좋아질 것 같았습니다.

원하시면 무료로 1페이지 진단을 해드릴게요.
제가 보는 기준은 소개 문구, 후기 배치, 가격/상담 CTA, 모바일 흐름입니다.
```

### 학원/클래스

```text
안녕하세요. 수업/클래스 정보를 보다가 연락드렸습니다.

체험수업이나 상담 신청 전 고객이 궁금해할 커리큘럼, 비용, 후기, 위치 정보가 한 페이지에 정리되면 문의가 더 쉬워질 것 같았습니다.

원하시면 현재 온라인 정보 기준으로 무료 홈페이지 구조 진단을 보내드리겠습니다.
```

### 뷰티/PT/필라테스

```text
안녕하세요. 인스타그램 콘텐츠를 보고 연락드렸습니다.

사진과 분위기는 좋은데, 처음 온 고객이 프로그램/가격/후기/예약 방법을 한 번에 확인할 수 있는 랜딩페이지가 있으면 예약 전환에 도움이 될 것 같았습니다.

원하시면 무료로 예약 전환 관점에서 개선 포인트를 정리해드릴게요.
```

## 홈페이지 개선 실험

| 실험                                                             | 변경 위치        | 가설                                           | 측정        |
| ---------------------------------------------------------------- | ---------------- | ---------------------------------------------- | ----------- |
| CTA를 `내 브랜드 진단 요청`에서 `무료 홈페이지 진단 받기`로 변경 | Hero, Contact    | 무료 진단 문구가 더 직접적이다                 | CTA 클릭    |
| 타겟별 진단 예시 추가                                            | Targets 아래     | 고객이 자기 업종을 발견하면 문의 가능성이 높다 | 스크롤/문의 |
| Before/After 샘플 추가                                           | Service 전       | 결과물을 상상하기 쉬우면 신뢰가 오른다         | 상담 전환   |
| 가격 가설을 더 명확히 공개                                       | Service Packages | 예산 불안을 줄이면 상담 품질이 좋아진다        | 견적 요청   |
| Notion 운영판 미리보기 추가                                      | AI Team 아래     | 운영 시스템이 차별점으로 보인다                | 체류/문의   |
