# Landing Brand Insight Card Template

기준일: 2026-05-27

목적: 리서치한 홈페이지, 랜딩페이지, 프리랜서 판매 페이지, 에이전시 사이트를 `랜딩페이지 효과`와 `브랜딩 이미지` 관점으로 정리해 Notion에 축적한다.

## Notion DB 이름

`Landing Brand Insight Cards`

## 속성

| Property                 | Type         | Description                                                          |
| ------------------------ | ------------ | -------------------------------------------------------------------- |
| Card Title               | Title        | 인사이트 카드 제목                                                   |
| Source URL               | URL          | 분석한 페이지 URL                                                    |
| Source Type              | Select       | Competitor, Freelancer, Agency, SaaS, Reference                      |
| Channel                  | Select       | 크몽, 숨고, 위시켓, 프리모아, Fiverr, Upwork, Webflow, Framer, 기타  |
| Target Segment           | Multi-select | 1인 전문가, 학원, 뷰티/PT, 로컬 서비스, 병원, B2B, 기타              |
| First Impression         | Rich text    | 방문자가 처음 느끼는 인상                                            |
| Above The Fold One-liner | Rich text    | 첫 화면에서 이해되는 한 문장                                         |
| Animation/3D Role        | Rich text    | 애니메이션/3D가 어떤 의미로 쓰였는지                                 |
| Trust Elements           | Multi-select | 후기, 숫자, 포트폴리오, 경력, 기술스택, 가격, 프로세스, 인증, 리포트 |
| CTA Flow                 | Rich text    | CTA가 어떤 순서와 문구로 이어지는지                                  |
| Landing Effect Score     | Number       | 전환 관점 점수, 1-5                                                  |
| Branding Image Score     | Number       | 브랜드 인상 점수, 1-5                                                |
| Our Copy Improvement     | Rich text    | 우리 홈페이지에 반영할 문구 개선안                                   |
| Our Section Improvement  | Rich text    | 우리 홈페이지에 반영할 섹션 개선안                                   |
| Our Visual Improvement   | Rich text    | 우리 홈페이지에 반영할 비주얼 개선안                                 |
| Instagram Content Idea   | Rich text    | 카드뉴스/릴스 아이디어                                               |
| Threads Content Idea     | Rich text    | 짧은 글/스레드 아이디어                                              |
| Blog Content Idea        | Rich text    | 블로그 글 아이디어                                                   |
| Status                   | Select       | Raw, Reviewed, Applied, Archived                                     |
| Captured Date            | Date         | 기록일                                                               |

## 예시 카드

```markdown
# 프리미엄 AI 랜딩페이지 판매 페이지 인사이트

Source URL: 예시 URL
Source Type: Freelancer
Channel: Fiverr
Target Segment: SaaS, 1인 창업자
Captured Date: 2026-05-27

## 1. 방문자가 처음 느끼는 인상

- 빠르고 기술적인 제작자처럼 보인다.
- 포트폴리오보다 기능과 납품 속도가 먼저 보인다.
- 고급 브랜드 이미지보다는 즉시 구매 가능한 상품 느낌이 강하다.

## 2. 첫 화면에서 이해되는 한 문장

> AI SaaS를 위한 고전환 랜딩페이지를 빠르게 제작한다.

판단:

- 대상과 결과가 명확하다.
- 다만 운영 시스템이나 지속 개선의 이미지는 약하다.

## 3. 애니메이션/3D가 의미 있게 쓰였는지

- 시각 효과가 있더라도 제작 역량을 보여주는 보조 장치에 가깝다.
- 서비스 운영 흐름을 설명하는 용도는 약하다.

## 4. 신뢰를 만드는 증거 요소

- 리뷰, 납품 수, 기술 스택, 패키지 가격.

부족한 점:

- 납품 후 운영과 개선 방식이 보이지 않는다.

## 5. CTA 흐름

현재 흐름:

1. 패키지 비교
2. 리뷰 확인
3. 구매 또는 문의

평가:

- 구매는 쉽지만, 고가 상담으로 끌고 가는 신뢰 설계는 약하다.

## 6. 우리 홈페이지 개선안

문구:

> 홈페이지 제작으로 끝내지 않고, 리서치와 콘텐츠가 계속 쌓이는 브랜딩 운영 시스템을 만듭니다.

섹션:

- "납품 후 매주 개선되는 것들" 섹션 추가.
- Research Library, Content Calendar, Weekly Report 예시를 한 줄 흐름으로 보여준다.

비주얼:

- 3D 장면은 로봇보다 리서치, Notion, 콘텐츠, 홈페이지가 연결되는 파이프라인으로 보여준다.

## 7. Instagram/Threads 콘텐츠화

Instagram:

- 카드뉴스: "예쁜 홈페이지와 팔리는 홈페이지의 차이 5가지"

Threads:

- "홈페이지 제작자는 많다. 그런데 납품 후 무엇이 쌓이는지 보여주는 곳은 적다."

Blog:

- 글: 랜딩페이지 제작보다 브랜딩 운영 시스템이 중요한 이유

## 점수

- Landing Effect Score: 4
- Branding Image Score: 3

## 다음 액션

- 우리 홈페이지에 "납품 후 운영" 섹션을 추가할지 실험한다.
```
