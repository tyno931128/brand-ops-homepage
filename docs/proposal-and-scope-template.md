# Proposal and Scope Template

기준일: 2026-05-28

이 문서는 `무료 진단 -> 7일 브랜딩 랜딩 -> 30일 운영 개선` 흐름에서 첫 유료 제안을 보낼 때 쓰는 제안서와 작업 범위 템플릿이다. 목적은 고객에게 명확한 기대치를 주고, 범위가 커져 수익성이 무너지는 일을 막는 것이다.

## 제안서 원칙

1. 제안서는 예쁜 소개서가 아니라 의사결정 문서다.
2. 목표는 하나로 좁힌다. 예: 체험수업 신청, 상담 신청, 예약, 견적 문의.
3. 범위와 제외 범위를 같이 쓴다.
4. 수정 횟수와 승인 시점을 명확히 한다.
5. 고객이 제공해야 하는 자료와 날짜를 적는다.
6. 추가 요청은 일정과 비용이 바뀐다는 점을 처음부터 설명한다.

## 7일 브랜딩 랜딩 제안서 구조

```text
제안명:
OOO를 위한 7일 브랜딩 랜딩페이지 제작

목표:
처음 보는 고객이 30초 안에 서비스, 신뢰 이유, 예약/문의 방법을 이해하게 만든다.

핵심 문제:
- 현재 정보가 인스타/블로그/플레이스/DM에 흩어져 있음
- 가격, 후기, FAQ, 예약 CTA가 한 흐름으로 보이지 않음
- 처음 보는 고객이 문의 전에 기본 정보를 다시 물어봐야 함

제안 방향:
- 첫 문장과 타겟을 명확히 정리
- 가격/프로그램/후기/FAQ/위치 정보를 한 페이지에 배치
- 모바일 기준으로 예약/상담 CTA를 고정
- 기본 SEO title/description과 검색 등록 체크리스트 제공
```

## 포함 범위

| 항목      | 포함 내용                                             |
| --------- | ----------------------------------------------------- |
| 전략      | 타겟, 고객 질문, 첫 문장, CTA 방향 정리               |
| 카피      | Hero, 문제 제기, 서비스/프로그램, FAQ, CTA 문구       |
| 디자인    | 1페이지 반응형 랜딩 디자인                            |
| 개발      | Next.js 기반 1페이지 구현                             |
| 섹션      | Hero, 문제, 서비스/프로그램, 신뢰 정보, FAQ, 문의 CTA |
| 기본 SEO  | title, description, Open Graph, 기본 구조화 데이터    |
| 로컬 체크 | 네이버/Google 등록 체크리스트 제공                    |
| 수정      | 2회 수정                                              |
| 배포      | Vercel 배포 가이드 또는 배포 대행                     |

## 제외 범위

| 제외 항목                        | 설명                                       |
| -------------------------------- | ------------------------------------------ |
| 로고/브랜드 아이덴티티 풀 패키지 | 별도 견적                                  |
| 사진 촬영/영상 제작              | 고객 제공 또는 별도 진행                   |
| 예약 시스템 자체 개발            | 네이버 예약, 카카오, 기존 폼 연결까지 기본 |
| 결제/회원/관리자 기능            | 별도 프로젝트                              |
| 다국어 페이지                    | 별도 견적                                  |
| 의료광고/법률 심의 대행          | 표현 검토는 가능하지만 법률 자문은 별도    |
| 무제한 수정                      | 포함 수정 2회 이후 추가 비용               |
| 장기 유지보수                    | 30일 운영 개선 또는 월 운영 패키지로 별도  |

## 7일 일정

| Day | 작업                       | 산출물                        |
| --- | -------------------------- | ----------------------------- |
| 1   | 자료 수집과 무료 진단 확정 | 자료 체크리스트, 첫 문장 방향 |
| 2   | 구조와 카피 초안           | 섹션 구성, Hero/CTA 문구      |
| 3   | 디자인 초안                | Desktop/Mobile 첫 화면        |
| 4   | 전체 페이지 디자인/개발    | 1차 개발본                    |
| 5   | 고객 피드백 반영 1차       | 수정본                        |
| 6   | 모바일/SEO/CTA 점검        | 검수본                        |
| 7   | 최종 수정과 배포           | 최종 링크, 운영 체크리스트    |

## 고객 제공 자료

프로젝트 시작 전 받아야 할 자료:

- 사업명/브랜드명
- 현재 링크: Instagram, 블로그, 네이버플레이스, 기존 홈페이지
- 프로그램/서비스 설명
- 가격 또는 가격 공개 여부
- 후기/사진/성과 자료
- 위치/운영시간/연락처
- 예약/상담 링크
- 자주 받는 질문 5개
- 참고하고 싶은 사이트 2~3개

## 수정 정책

포함 수정:

- 1차 구조/카피 수정
- 1차 디자인/모바일 수정

추가 비용이 필요한 요청:

- 새로운 섹션 추가
- 새로운 페이지 추가
- 전체 톤 변경
- 고객 제공 자료 지연으로 인한 일정 재조정
- 예약/결제/회원/관리자 등 기능 추가
- 포함된 수정 횟수 이후 추가 수정

## 가격 가설

| 상품            |            가격 | 설명                                 |
| --------------- | --------------: | ------------------------------------ |
| 무료 진단       |             0원 | 첫 문장, 빠진 정보, CTA 진단         |
| 7일 브랜딩 랜딩 |   79만~149만 원 | 1페이지 제작, 기본 SEO, CTA 정리     |
| 리뉴얼 패키지   |  180만~300만 원 | 기존 홈페이지 구조 재설계, 섹션 확장 |
| 30일 운영 개선  | 월 39만~79만 원 | 콘텐츠 소재, CTA, 랜딩 문구 개선     |

초기에는 첫 5팀 한정으로 가격을 낮출 수 있지만, 범위는 줄이지 말고 명확히 제한한다.

## 결제 조건 예시

```text
총액: 000만 원

결제:
- 착수금 50%: 일정 확정 및 자료 수집 시작 전
- 잔금 50%: 최종 배포 또는 파일 인도 전

포함 수정:
- 구조/카피 수정 1회
- 디자인/모바일 수정 1회

추가 요청:
포함 범위를 벗어난 요청은 별도 견적과 일정 승인 후 진행합니다.
```

## 상담 후 제안 메시지

```text
오늘 이야기 기준으로 보면, 지금 가장 먼저 정리할 것은 큰 홈페이지가 아니라 고객이 예약 전에 확인하는 정보 흐름입니다.

제안 범위는 7일 브랜딩 랜딩 1페이지입니다.

핵심은 다음 3가지입니다.

1. 처음 보는 고객이 바로 이해할 첫 문장
2. 가격/프로그램/후기/FAQ 같은 신뢰 정보
3. 체험수업/상담/예약으로 이어지는 CTA

결제나 회원 기능처럼 복잡한 개발은 이번 범위에 넣지 않고, 네이버 예약/카카오/기존 폼으로 연결하는 방식이 맞아 보입니다.
```

## 리스크 문구

```text
검색 순위, 문의 수, 매출 증가를 보장하지는 않습니다.

다만 고객이 이해하고 신뢰하고 문의하기 위해 필요한 첫 문장, 정보 구조, CTA, 기본 검색 등록 체크리스트를 정리합니다.
```
