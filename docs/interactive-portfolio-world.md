# Interactive Portfolio World

기준일: 2026-05-28

이 문서는 Brand Ops 홈페이지에 추가한 3D 인터랙티브 포트폴리오 월드의 의도와 구현 기준을 정리한다.

## 방향

참고 사이트는 Bruno Simon의 3D 포트폴리오다. 그대로 복제하지 않고, 다음 패턴만 Brand Ops에 맞게 재해석한다.

- 방문자가 직접 움직이는 경험
- 공간 안의 구역을 열어보는 포트폴리오 구조
- 게임 같은 첫인상
- 작업물, 제작 방식, 문의 CTA를 한 공간 안에서 탐색하는 흐름

## 현재 구현

- 경로: `/studio`
- 컴포넌트: `src/components/interactive-portfolio-world.tsx`
- 조작: WASD, 방향키, 마우스/터치 이동
- 구역: 만든 사이트, 무료 진단, AI 팀 운영, 7일 랜딩
- 접근성: 버튼으로도 모든 구역을 열 수 있고, 3D 영역에는 조작 안내와 `role="application"`을 둔다.

## 확장 계획

1. 실제 고객 작업물이 생기면 `만든 사이트` 구역을 프로젝트별 게이트로 바꾼다.
2. `/projects`의 샘플 카드를 실제 사례 페이지로 분리한다.
3. 모바일에서는 3D 조작보다 버튼 탭과 짧은 요약이 우선 보이게 한다.
4. 3D 월드가 느려지면 첫 화면이 아니라 `/studio` 전용 경험으로만 유지한다.

## 주의

- Bruno Simon 사이트의 자동차, 맵, 그래픽을 복제하지 않는다.
- Brand Ops의 캐릭터, 색, 패드, 구역 구조는 자체 자산으로 유지한다.
- 실제 고객 사례가 생기기 전까지 `/projects`는 샘플임을 명확히 표시한다.
