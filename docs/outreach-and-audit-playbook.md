# Outreach and Audit Playbook

기준일: 2026-05-28

이 문서는 AI 브랜딩 홈페이지 제작 사업의 첫 매출을 만들기 위한 리드 선별, 무료 진단, DM, 상담 흐름 기준이다. 핵심은 `홈페이지가 없나요?`라고 묻는 것이 아니라, **이미 온라인에서 고객을 모으고 있지만 문의/예약 전환 구조가 약한 곳**을 찾아 구체적인 문제를 짚는 것이다.

## 핵심 원칙

1. 홈페이지가 전혀 없는 곳보다, 온라인 마케팅을 이미 믿는 곳을 우선한다.
   인스타, 네이버플레이스, 블로그, 광고, 예약 링크가 있는데 전환 흐름이 약한 사업자가 더 구매 가능성이 높다.

2. 기술 문제가 아니라 사업 결과로 말한다.
   “모바일 UX가 안 좋습니다”보다 “체험수업 가격과 예약 버튼을 처음 보는 사람이 찾기 어렵습니다”가 낫다.

3. 무료 진단은 짧고 구체적이어야 한다.
   산출물은 `첫 문장 후보 1개`, `빠진 신뢰 정보 5개`, `문의 CTA 구조 1개`로 제한한다.

4. 바로 큰 홈페이지를 팔지 않는다.
   첫 제안은 `무료 진단 -> 7일 브랜딩 랜딩 -> 30일 운영 개선`으로 이어간다.

5. 외부 연락은 기록 없이 하지 않는다.
   리드 후보, 보낸 문구, 반응, 다음 액션은 Notion Leads DB 또는 로컬 문서에 남긴다.

## 리드 점수표

총점 10점 이상이면 무료 진단 제안 대상, 14점 이상이면 우선 연락 대상이다.

| 항목                           | 점수 | 기준                                            |
| ------------------------------ | ---: | ----------------------------------------------- |
| 인스타/블로그/플레이스 운영 중 |   +3 | 콘텐츠나 리뷰가 이미 쌓여 있다                  |
| 홈페이지 없음 또는 오래됨      |   +3 | 공식 링크가 없거나 모바일에서 보기 어렵다       |
| 예약/상담/견적이 매출과 직결   |   +3 | 체험수업, 예약, 상담, 견적 문의가 있다          |
| 객단가가 낮지 않음             |   +2 | 패키지, 회차권, 상담, 프로그램 단가가 있다      |
| 후기/사진/성과 자료가 있음     |   +2 | 랜딩페이지에 옮길 신뢰 재료가 있다              |
| 가격/프로그램/FAQ가 흩어짐     |   +2 | 고객이 문의 전에 확인할 정보가 분산되어 있다    |
| 지역 검색 니즈가 있음          |   +1 | 지역명 + 업종 검색으로 고객이 들어온다          |
| 의료/법률/금융 표현 리스크     |   -2 | 검토 없이 카피를 쓰기 어렵다                    |
| 기능 개발 요구가 큼            |   -3 | 예약 시스템, 결제, 회원, 관리자 개발을 요구한다 |
| 온라인 마케팅 의지가 낮음      |   -3 | 웹사이트 필요성을 느끼지 못한다                 |

## 우선 연락 대상

1. 필라테스/PT/피부관리
   인스타 콘텐츠는 많은데 체험수업, 가격, 후기, 예약 CTA가 흩어진 곳.

2. 학원/클래스/공방
   블로그와 공지는 있는데 커리큘럼, 후기, 상담 신청이 한 페이지로 정리되지 않은 곳.

3. 1인 전문가/강사/코치
   콘텐츠는 쌓였지만 서비스 설명, 패키지, 상담 CTA가 명확하지 않은 곳.

4. 로컬 서비스
   작업 사례와 후기는 있지만 견적 문의 폼, 지역 정보, 작업 과정이 약한 곳.

5. 병원/상담센터
   정보 정리 니즈는 크지만 의료광고·상담 표현 리스크가 있어 2단계로 접근한다.

## 무료 진단 구성

```text
1. 첫 화면 진단
- 지금 첫 문장으로 고객이 무엇을 이해하는가?
- 누구를 위한 서비스인지 보이는가?
- 다음 행동이 하나로 보이는가?

2. 신뢰 정보 진단
- 가격/프로그램/후기/FAQ/위치/운영 시간이 충분한가?
- 고객이 문의 전에 불안해할 질문에 답하는가?
- 전문성 증거가 너무 아래에 있거나 흩어져 있지 않은가?

3. 문의 동선 진단
- 전화, 카카오, 예약폼, DM 중 무엇을 눌러야 하는가?
- 모바일에서 CTA가 보이는가?
- SNS에서 들어온 고객이 홈페이지에서 이탈하지 않는가?
```

## DM 템플릿

### 필라테스/PT

```text
안녕하세요. 인스타와 예약 흐름을 보다가 연락드렸습니다.

사진과 분위기는 좋은데 처음 보는 고객이 체험수업, 가격, 후기, 예약 방법을 한 번에 확인하기는 조금 어려워 보였습니다.

원하시면 무료로 1페이지 진단을 보내드릴게요. 첫 문장, 빠진 신뢰 정보, 예약 CTA 기준으로 바로 고칠 3가지만 정리해드립니다.
```

### 피부관리/뷰티

```text
안녕하세요. 관리 프로그램과 콘텐츠를 보고 연락드렸습니다.

처음 방문하는 고객 입장에서는 대표 코스, 가격대, 후기, 위치, 예약 방법이 한 화면에 정리되면 예약 결정이 더 쉬울 것 같았습니다.

원하시면 현재 인스타/링크 기준으로 무료 진단을 드리겠습니다. 바로 고칠 첫 화면 문구와 예약 동선 중심으로 봅니다.
```

### 학원/클래스

```text
안녕하세요. 수업 정보를 보다가 연락드렸습니다.

커리큘럼과 후기, 체험수업/상담 신청이 한 페이지에 정리되면 학부모나 수강생이 비교하기 더 쉬울 것 같았습니다.

원하시면 무료로 홈페이지 구조 진단을 보내드리겠습니다. 첫 문장, 빠진 정보, 상담 CTA 3가지만 짧게 정리합니다.
```

### 1인 전문가/코치

```text
안녕하세요. 콘텐츠를 보고 연락드렸습니다.

전문성은 잘 보이는데 처음 보는 사람이 어떤 문제를 맡기면 되는지, 어떤 상담/패키지로 이어지는지 한 번에 이해하기는 조금 어려워 보였습니다.

원하시면 무료로 개인 브랜딩 랜딩 구조를 짧게 진단해드릴게요. 첫 문장, 신뢰 증거, 상담 CTA 기준으로 봅니다.
```

### 로컬 서비스

```text
안녕하세요. 작업 사례와 후기를 보고 연락드렸습니다.

지역 고객이 견적을 비교할 때 작업 과정, 가격 기준, 후기, 문의 방법이 한 페이지에 정리되어 있으면 신뢰를 얻기 더 쉬울 것 같았습니다.

원하시면 현재 링크 기준으로 무료 진단을 보내드리겠습니다. 견적 문의를 막는 부분 3가지만 짧게 정리합니다.
```

## 진단 회신 형식

```text
안녕하세요. 요청하신 무료 진단을 간단히 정리했습니다.

1. 첫 문장
현재:
제안:

2. 빠진 신뢰 정보
-
-
-
-
-

3. 문의 동선
현재:
제안:

바로 고칠 1순위:

작게 시작한다면:
7일 브랜딩 랜딩으로 첫 화면, 가격/후기/FAQ, 문의 CTA를 한 페이지로 정리하는 방식이 맞아 보입니다.
```

## 상담 질문

1. 지금 문의는 어디에서 가장 많이 오나요?
2. 문의 전 고객이 가장 많이 묻는 질문은 무엇인가요?
3. 가격이나 프로그램 정보를 공개하기 어려운 이유가 있나요?
4. 예약, 상담, 견적 중 가장 늘리고 싶은 행동은 무엇인가요?
5. 기존 고객 후기나 사진 자료는 얼마나 있나요?
6. 7일 안에 한 페이지로 정리한다면 반드시 들어가야 할 정보는 무엇인가요?

## 제안 구조

```text
목표:
처음 보는 고객이 30초 안에 서비스, 신뢰 이유, 문의 방법을 이해하게 만든다.

범위:
- 1페이지 브랜딩 랜딩
- 첫 문장/카피 정리
- 가격/프로그램/후기/FAQ 구조
- 모바일 CTA
- 기본 SEO title/description

일정:
자료 수집 1일 -> 카피/구조 2일 -> 디자인/제작 3일 -> 수정 1일

다음 단계:
30일 동안 문의 반응을 보고 문구, CTA, 콘텐츠 주제를 개선한다.
```

## 주의할 표현

- `무조건 문의가 늘어납니다` 금지
- `매출 2배 보장` 금지
- 의료/상담 업종에서 치료 효과, 전후 비교, 후기 중심 설득 금지
- 실제 고객이 아닌 샘플을 실제 사례처럼 표현 금지
- “AI가 자동으로 다 해준다”보다 “AI가 리서치와 정리를 돕고 사람이 설계한다”로 표현
