# Security Policy

## Supported Scope

This repository currently supports the latest `main` branch only.

## Reporting

Do not open a public issue for secrets, authentication problems, private data
exposure, or exploitable vulnerabilities.

Instead, contact the maintainer privately through the contact method listed in
the public website or repository profile.

## Rules for Contributors

- Never commit `.env*` files.
- Never commit Notion tokens, OpenAI API keys, Vercel tokens, OAuth secrets, or
  customer data.
- Redact private client details in screenshots, examples, and fixtures.
- Keep publish/posting actions behind explicit human approval.
